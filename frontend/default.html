Set nilch as your default search engine

Most browsers on mobile currently do not let you set a default search engine besides a few pre-set options, so this is mostly for desktop. However, you can set nilch as your default search engine on Firefox mobile.

Google Chrome (and other Chromium-based browsers)

  • Enter your browser settings
  • Go to the Search engine section (you may need to first press the three lines in the top left corner)
  • Click Manage search engines and site search
  • Scroll down to Site search and click Add
  • Type nilch in the name and shortcut fields
  • Type https://nilch.org/results?q=%s in the URL field and click Add
  • Click the three dots symbol next to the nilch item in the Site search list
  • Click Make default

Firefox

  • Enter your browser settings
  • Go to the Search section (it may just be a magnifying glass icon)
  • Scroll down to Search Shortcuts and click Add
  • Type nilch in the Search engine name field
  • Type https://nilch.org/results?q=%s in the URL field and click Save Engine
  • Scroll back up to the start of the Search section, click the Default Search Engine dropdown (it will likely currently say Google, and click nilch
